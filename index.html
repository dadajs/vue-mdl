Quick start

# Clone this repo
$ git clone git@github.com:dadajs/vue-mdl-starter.git

# Install dependencies
$ cd vue-mdl-starter
$ npm install

# Fire up webpack-dev-server
$ npm run dev

# Build to /dist
$ npm run build
